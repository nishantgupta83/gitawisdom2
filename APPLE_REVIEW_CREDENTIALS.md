## Testing Instructions for Apple Reviewers

### How to Test Journal Feature:

1. **Launch the GitaWisdom app**

2. **Navigate to Journal:**
   - Tap the "Journal" tab in the bottom navigation bar (4th icon from left)

3. **Sign In:**
   - You will see an authentication prompt
   - Tap the blue "Sign In" button
   - Enter the credentials above
   - Tap "Sign In" to authenticate

4. **Access Journal:**
   - Once signed in, you'll have full access to the Journal feature
   - Journal entries sync across devices via cloud storage
   - You can create, edit, and delete journal entries

### Alternative Testing Option:

**Guest Mode (No Account Required):**
- From the Journal authentication prompt, tap "Continue as Guest"
- Provides local-only journal access without creating an account
- Data stored only on device (no cloud sync)
- Privacy-focused option for users who don't want to create accounts
